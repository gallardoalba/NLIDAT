# NLIDAT: Non-Linear Dynamics Analysis Toolbox

NLIDAT extends [TSENAT](https://github.com/gallardoalba/TSENAT) with non-linear dynamics methods to analyze isoform complexity in RNA-seq data through Tsallis entropy. It provides tools to identify chaotic splicing patterns, detect critical transitions and distinguish biological chaos from noise.
